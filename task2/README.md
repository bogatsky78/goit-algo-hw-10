# Завдання 2 — Обчислення інтеграла методом Монте-Карло

## Функція

```
f(x) = sin(x),  x ∈ [0, π]
```

## Метод

Генерується 100 000 випадкових точок у прямокутнику `[0, π] × [0, 1]`.  
Частка точок, що потрапила під криву, множиться на площу прямокутника — це і є наближення інтеграла.

## Результати

| Метод        | Результат  | Похибка    |
|--------------|------------|------------|
| Монте-Карло  | 2.000032   | 0.000032   |
| scipy quad   | 2.000000   | 0.000000   |
| Аналітично   | 2.000000   | —          |

Аналітичний результат: `∫₀^π sin(x) dx = [-cos(x)]₀^π = -cos(π) + cos(0) = 1 + 1 = 2`

## Висновок

Метод Монте-Карло дав результат **2.000032** при N = 100 000 точок, що відрізняється від точного значення **2** лише на **0.000032** (~0.0016%).

`scipy.integrate.quad` повертає практично точний результат із машинною похибкою, тому виступає еталоном для порівняння.
